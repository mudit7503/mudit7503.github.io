Aeroderivative Turbine Power Control ↗ GitHub

02

PWM-Controlled 4WD RC Vehicle

  • Designed and assembled an RC car as part of a student team, using Arduino, two BTS7960 motor drivers, and Johnson geared DC motors to achieve four-wheel drive motion.
  • Programmed the microcontroller using Arduino IDE, implementing PWM-based speed control and direction logic to drive the motor drivers.
  • Interfaced the Arduino with an off-the-shelf RC receiver to translate transmitter inputs into vehicle-side control signals, enabling coordinated multi-motor control, smooth steering, and effective obstacle navigation.
  • Tested, debugged, and refined firmware and hardware during live obstacle-course runs, improving motor response, stability, and overall system reliability.
